细菌 Quorum Sensing (QS)Network 的研究
2012 - 2016
  • 用优化误差函数的方法以实验数据拟了 QS network 的参数。

用 Agent Based Model 研究 Biofilm
2012 - 2016
  • P.aerugionsa biofilm 会产生两类高分子物质(Pel, Psl)维持其形状。Prof Yang Liang 的实 验组通过实验现象提出 Pel 和 Psl 的模型(Hypothesis)。
  • 建立 ABM 预测只产生 Pel 和 Psl 的突变种组成的 biofilm 在不同养料和水流条件下的形状和突变种的数量比例。其中包括对 self-avoidant agent (互不重叠), 快速扩散反应，慢速扩散反应和 biofilm detachment 过程的数学建模。
  • 模型得出的 Psl+Pel+ 和 Psl-Pel+ 混合生长结果与实验吻合，验证了 Psl++ 菌群合并的趋势，并得出了 Psl++ 的适应性曲线。
